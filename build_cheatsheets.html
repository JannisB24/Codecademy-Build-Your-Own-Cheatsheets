HTML Tables Cheatsheet


Table Tags:

Tag Name Description
<table>
</table>		 Table To Create a Table in a HTML Document
<tr>
</tr>		 Table-Rows To Create Table-Rows in a Table
<td>
</td>		 Table-Data To Write Data in a Table-Row
<th>
</th>		 Table-Headings To Make Headings for the Table-Rows
<tbody>
</tbody>		 Table-Body To Organize the Table: Here is the Data
<thead>
</thead>		 Table-Head To Organize the Table: Here are the Headings
<tfoot>
</tfoot>		 Table-Footer To Summarize or Condlude Data from the Table in a row

Table Attributes:

Attribute Name Description
table {
border: 1px solid black;
}		 Border To Create Borders for the Table in the CSS File
Border-Styles: Solid, Dotted, None
<td colspan="2">Some Data</td> Colspan To Use multiple Columns instead of One
<td rowspan="2">Some Data</td> Colspan To Use multiple Rows instead of One
<th scope="row">Some Heading</th> Scope To Specify if a Heading is for Rows or for Columns
